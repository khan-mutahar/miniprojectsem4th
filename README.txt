Scene Text Detection on Video
# Scene Text Detection Project

## Description
This project detects text from images/video using OpenCV and Tesseract OCR.

## Features
- Detects text from video frames
- Uses pytesseract for OCR

Steps to run:
1. Install requirements: pip install -r requirements.txt
2. Install Tesseract OCR separately
3. Place your video file as 'video.mp4'
4. Run: python main.py
